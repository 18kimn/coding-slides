welcome to the coding-slides backend!

Overview

This is a super simple API for educators. It provides access to a simple server resource to run code interactively online. The main intention is to be used in slides, as described here, but it can also be used for teaching APIs, in other web apps besides Reveal.js slides, or in general just as an experimental playground for running code. Currently supports R, python3, and nodeJS.

This API is hosted on a single AWS EC2 instance, which is a virtual machine running Ubuntu that simply runs the given code and sends it back to the client. This is dangerous sort of by nature, and although I have restricted the permissions of the API to be unable to create or destroy files, it's possible that the server may be misused. Also, because this is running on a single EC2 t2-micro machine, I'm predicting the capacity will be fairly low and it will break with any real usage. All of that to say, please be courteous and kind, only use this API for your own slides and quick demos, and don't run anything intensive or dangerous here. :)

Usage

    Endpoints:
  • / (the root): this page, an intro to the backend
  • /src : the main endpoint to run an execute scripts. Receives GET requests; returns stdout and stderror as strings.
      Query options:
    • data : A URI-encoded string representing the code to run
    • languageEngine : The name of an executable script to use to run the code input. Currently the only supported options are Rscript, python3, and node.
